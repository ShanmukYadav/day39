# Week 07 - Day 39 Assignment (NLP)

## Overview
This project focuses on handling complex NLP patterns in Indian e-commerce reviews and building sentiment/aspect classifiers.

## Tasks Covered
### Q1:
Handled 5 NLP challenges:
- Negation
- Sarcasm
- Code-mixing
- Implicit sentiment
- Comparative statements

Includes preprocessing + baseline failure cases.

### Q2:
- Explained why aspect-level classification is harder
- Improved approach suggestions
- Extracted aspect-sentiment pairs
